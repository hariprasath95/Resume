# Resume — Hariprasath Parthasarathy

LaTeX source and compiled PDF of my resume, targeting senior software engineering roles in database internals and distributed systems.

## Files

- `resume.tex` — LaTeX source (single-column, ATS-friendly layout)
- `resume.pdf` — compiled output

## Build

```sh
pdflatex resume.tex
```

The summary section in `resume.tex` includes commented one-line variants for tailoring to specific companies — append one to the summary and rebuild.
